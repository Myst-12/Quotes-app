# Quotes App  
A responsive and minimal quotes application that fetches and displays inspirational quotes using a public API. Built with modern web technologies, it provides a smooth and clean user experience.

## Features  
- Fetch quotes from an API  
- Display multiple quotes dynamically  
- Simple and clean UI  
- Fast and responsive design  
- Error handling for API failures  

## Tech Stack  
- Frontend: HTML, CSS, JavaScript (or React if applicable)  
- API: ZenQuotes API  
- Styling: CSS / Tailwind / Bootstrap  
- Deployment: GitHub Pages / Vercel / Netlify

## Installation & Setup  

Clone the repository:  
```bash
git clone https://github.com/your-username/quotes-app.git
```

Navigate to the project folder:  
```bash
cd quotes-app
```

Install dependencies (if applicable):  
```bash
npm install
```

Run the project:  
```bash
npm start
```

## API Used  
ZenQuotes API provides a collection of inspirational quotes that can be fetched and displayed dynamically.  

Endpoint used:  
```
https://zenquotes.io/api/quotes
```
